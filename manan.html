MANAN

Manan is a coding-focused society that brings together students interested in software development, programming, and computer science. It offers coding bootcamps, hackathons, coding competitions, and mentorship programs to help members improve their coding skills and stay up-to-date with the latest technologies. Manan encourages innovation and collaboration in the coding community.
